損益計算書の4つの利益：比較と分析ポイント完全ガイド

  • #損益計算書
  • #利益比較
  • #経営分析
  • #財務諸表

損益計算書（P/L）には段階に応じて複数の「利益」が登場します。それぞれの利益は、企業の異なる側面（商品力、営業力、財務体質など）を映し出しています。

以下の表は、主要な4つの利益について「計算式」「意味」「分析の視点」「チェックポイント」を整理したものです。

売上総利益

意味
商品・製品自体の儲け
計算式
売上高
－ 売上原価
分析の視点
原価管理ができているかが分かる
Check Point
安売りしていないか？

営業利益

意味
本業全体の儲け
計算式
売上総利益
－ 販管費
分析の視点
営業力・コスト力が分かる
Check Point
本業が儲かっているか？

経常利益

意味
会社全体の通常利益
計算式
営業利益
＋ 営業外収益
－ 営業外費用
分析の視点
財務体質が分かる
Check Point
借金が重くないか？

当期純利益

意味
最終結果の儲け
計算式
税引前純利益
－ 法人税等
分析の視点
経営の最終成果
Check Point
黒字倒産していないか？

活用ポイント

この表を横に見るだけでなく、以下のような視点で比較すると企業の課題が見えてきます。

粗利はあるが、営業利益がない場合

「商品は良い（売上総利益はプラス）が、売り方や管理コストに問題がある（販管費が高い）」という状態です。広告宣伝費のかけ過ぎや、人件費の高騰を疑う必要があります。

営業利益はあるが、経常利益がない場合

「本業は順調だが、借金の利息支払いが重い」という財務体質の問題が考えられます。過去の過剰な設備投資が負担になっているケースなどが該当します。
