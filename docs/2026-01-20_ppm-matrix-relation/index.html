PPMとマトリクスの関係

PPM（プロダクト・ポートフォリオ・マネジメント）を上位概念として、BCG・GE・ADL・技術ポートフォリオがどのように位置づけられるのか、その包含関係と特徴を解説します。

# 1. 全体像：PPMと各マトリクスの関係

PPM (Product Portfolio Management)

複数事業・製品・技術を俯瞰し、
資源配分・投資判断を行う「思想・枠組み」

ポイント： PPMという「考え方」を、目的に応じて具体化した「道具」がこれらのマトリクスです。

3. ADLマトリクス

縦：競争地位 横：産業ライフサイクル
導入成長成熟衰退
  • 事業の「時間軸（寿命）」を考慮できる。
  • 撤退のタイミング判断に強い。

# 3. 比較まとめ

手法 特徴・用途 キーワード
BCG 最もシンプル。キャッシュフロー配分。 花形・金のなる木

技術PF 研究開発テーマの選定・資源配分。 R&D・成熟度

試験・実務で使える一言まとめ

「PPMとは、複数の事業・製品・技術をポートフォリオとして管理する考え方であり、BCG・GE・ADL・技術ポートフォリオは、その目的や分析対象に応じた具体的手法である。」
