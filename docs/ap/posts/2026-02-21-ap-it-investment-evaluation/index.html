IT投資効果の評価指標（PBP・IRR・NPV・ROI）

#応用情報 #ITストラテジ #投資評価 #NPV #IRR

「このシステム投資はやるべきか？」を判断するための4つの代表的な指標（PBP、NPV、IRR、ROI）の違いと特徴を図解します。

1. PBP（Pay Back Period：回収期間法）

「元が取れるまで何年か？」を測る、一番シンプルで直感的な指標です。
初期投資によるキャッシュアウト（支出）を、その後の利益で何年後に回収できるかを見ます。

2. 現在価値（Present Value）の考え方

NPVとIRRを理解する上で欠かせないのが「現在価値」です。
「今の100万円」と「3年後の100万円」は価値が違います（利息がつくため、今持っている方が価値が高い）。将来入ってくるお金を「今の価値（現在価値）」に割り引いて（小さくして）計算するのが基本です。

3. NPV と IRR の違い

現在価値の考え方を使った2つの指標を比較します。「金額」で出すか「％（利率）」で出すかの違いです。

NPVとIRRの判断基準の違い
NPV (Net Present Value / 正味現在価値) 単位：金額（円）
将来の利益を現在価値に直し、そこから初期投資額を引いた「純粋な儲けの金額」。

✅ 判断基準:
NPV ＞ 0 なら投資OK！
（例：110万 - 100万 = +10万円）
IRR (Internal Rate of Return / 内部収益率) 単位：利率（％）
投資額と、将来利益の現在価値が「ピッタリ同じ（NPV=0）」になる時の「実質的な利回り」。

✅ 判断基準:
IRR ＞ 要求利回り なら投資OK！
（例：この投資は年利12%の価値がある）

4. ROI（Return On Investment：投資利益率）

「投資額に対して何％の利益が出たか」を示す、ビジネスで最もよく使われる指標の一つです。（現在価値などの複雑な計算は含みません）

ROIの割合イメージ
投資額 (100%) : 100万円
利益: 20万円
ROI ＝ ( 20万 ÷ 100万 ) × 100 ＝ 20%

「いくら儲かったか」ではなく「何％儲かったか（効率の良さ）」を見る。

まとめ：試験対策用比較表

試験では「どの指標が何を意味するか」と「単位」が問われます。

指標 名称 意味・特徴 単位 投資OKの基準
PBP 回収期間法 投資額を何年で回収できるか 年・月 自社の基準期間より短い
NPV 正味現在価値 現在価値ベースでいくら儲かるか 金額 (円) NPV ＞ 0
IRR 内部収益率 この投資の実質的な利回り (NPVが0になる割引率) 利率 (％) IRR ＞ 要求利回り
ROI 投資利益率 投資額に対する利益の割合 (効率性) 割合 (％) 自社の目標値より高い
