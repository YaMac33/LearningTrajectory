集合演算の記号とベン図（空集合・全体集合・積・和・補・差・対称差）

#基礎理論 #数学 #集合 #基本情報

アルゴリズムやデータベースの基礎となる「集合」。それぞれの演算記号の読み方と、視覚的なイメージ（ベン図）をセットで覚えましょう。

1. 集合の基本となる「枠組み」と「空っぽ」

集合を考える上で、最初に定義される「全体」と「何もない状態」です。

Ω 全体集合（オメガ）
その問題で「考える範囲のすべて」。
確率統計の分野では「標本空間」とも呼ばれます。
図で表す時の「一番外側の四角い枠」です。
∅ / φ 空集合（くうしゅうごう / ファイ）
要素が「1つもない」集合。
中身が空っぽの箱 { } を意味します。（※ ∅ が正式な記号ですが、文脈で φ も使われます）

2. 5つの主要な集合演算（ベン図）

青く塗られている部分（#の結果領域）が、その演算が表す範囲です。

積集合 (Intersection) A ∩ B エー キャップ ビー
AとBの両方に入る要素。
（共通部分・重なり）
和集合 (Union) A ∪ B エー カップ ビー
AまたはBのどちらかに入る要素。
（両方含む・全部）
補集合 (Complement) AC, A エー シー / エー バー
全体Ωの中で、Aに入っていない部分。
（Aの外側すべて）
※「Ω \ A」と書くこともあります。
差集合 (Difference) A \ B エー バックスラッシュ ビー
Aには入っていて、Bには入っていない部分。
（Aから重なりを除いた左側の三日月）
※ A \ B と B \ A は別の形になります。
対称差集合 (Symmetric diff.) A △ B (または A ⊕ B) エー デルタ ビー
AとBのどちらか片方にだけ入る要素。（共通部分は除く、論理演算のXORと同じ）
(A \ B) ∪ (B \ A)

3. 記号と読み方の早見表

情報処理試験や数学のテキストで迷ったときのためのまとめです。

役割 記号 読み方
空集合 ∅ / φ くうしゅうごう / ファイ
全体集合 Ω オメガ
積集合 ∩ キャップ（帽子の形）
和集合 ∪ カップ（コップの形）
補集合 AC, A, Ω \ A シー / バー / バックスラッシュ
差集合 \ (A−Bとも書く) バックスラッシュ（A 差 B）
対称差 △, ⊕ デルタ / オープラス (XOR)
