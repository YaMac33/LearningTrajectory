クラスター分析とは？他の多変量解析手法との違い

#データサイエンス #機械学習 #多変量解析 #基本情報 #応用情報

「この顧客とあの顧客は似ているな」「この商品とあの商品は同じグループだ」。このように、あらかじめ正解（ラベル）がないデータの中から、自然なグループを見つけ出す手法が「クラスター分析」です。

【よくある問題パターン】
「あらかじめ分類の基準が与えられていないデータ群から、データ間の類似度に基づいて似たもの同士を集め、いくつかのグループを作る手法はどれか？」

➡ 答え：クラスター分析

1. クラスター分析の仕組みとイメージ

分析対象となる個々のデータ（点A〜I）が持っている特徴（変数の値）を比較し、数学的な「距離」が近い（似ている）もの同士をまとめていきます。

バラバラのデータを「群（クラスター）」に分ける
元のデータ
(分類されていない状態)
類似度で集める ➡
クラスター
(似たもの同士のグループ)

2. 4つの多変量解析手法のまとめ

情報処理試験で頻出の「主成分分析」「因子分析」「回帰分析」、そして今回の「クラスター分析」。これらは「データのどこに注目しているか（目的）」で整理するとスッキリします。

主成分分析 注目：変数の統合
[ 多 ] ➡ [ 一 ]
複数の変数（身長、体重など）が持つ情報を凝縮し、少数の新しい変数（体格スコアなど）に減らす。
因子分析 注目：背後の原因
[ 因 ] ➡ [ 多 ]
複数の変数に共通して影響を与えている「目に見えない背後の原因（共通因子）」を探り出す。
回帰分析 注目：変数間の因果
[ Ｘ ] ➡ [ Ｙ ]
ある変数（説明変数）から、別の変数（目的変数）の値を予測するための数式（モデル）を作る。
クラスター分析 注目：個体の類似
[ バラバラ ] ➡ [ 群 ]
変数ではなく「サンプル（個体）」を比較し、似たもの同士をまとめてグループ化する。
💡 最大の違い：「変数」か「個体（サンプル）」か
前の3つ（主成分・因子・回帰）は、主に「項目（変数）」の関係性に注目して計算を行います。
一方、クラスター分析は「データ1件1件（個体）」の関係性に注目し、顧客や商品を分けるために使われます。

試験対策まとめ：キーワードで見抜く

手法 主な目的 試験でのキーワード
クラスター分析 似たもの同士をグループ化する 類似度、距離、あらかじめ分類基準がない、群
主成分分析 変数を減らして要約する 次元削減、合成変数、情報の損失を最小に
因子分析 相関の理由を推定する 背後にある要因、共通因子、心理テスト
回帰分析 未知の数値を予測する 説明変数、目的変数、最小二乗法
