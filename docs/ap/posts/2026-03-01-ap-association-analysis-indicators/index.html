アソシエーション分析と3つの評価指標

#データマイニング #基本情報 #応用情報 #統計

「おむつを買う人は、ビールも一緒に買いやすい」といった、データの中に潜む「同時に起きやすいパターンの組合せ（ルール）」を発見する手法をアソシエーション分析と呼びます。小売業のレシート分析（バスケット分析）などでよく使われます。

1. 評価するための3つの重要指標

「Aを買う人は、Bも買う」というルール（{A} → {B}）が、本当に価値のあるルールなのかを評価するために、3つの指標を使います。

支持度 Support（サポート）
AとBを同時に含む件数 全取引の件数
その組合せが全体の中でどれくらい出現するか（全体の何％か）を表します。これが低すぎるルールは「たまたま起きたレアケース」として除外します。
確信度 Confidence（コンフィデンス）
AとBの同時出現 Aの出現件数
「Aを買った」という条件のもとで、Bも買う条件付き確率です。これが高いほど「Aを買えばBも買われやすい」と言えます。
リフト値 Lift（リフト）
確信度 ( P(B|A) ) B全体の出現率 ( P(B) )
Aを買ったときのBの確率が、Bが普段売れている確率と比べて何倍に跳ね上がるか。偶然以上の強い関係があるかを測ります。

2. ベン図で見る指標のイメージ

数式だけだと混乱しやすいため、面積（ベン図）でイメージを持っておくと試験で間違えにくくなります。

全体・Aの出現・Bの出現・同時出現の関係
  • 支持度： 四角形(全体)の中で、緑の部分(AかつB)が占める割合
  • 確信度： 青い円(A)の中で、緑の部分(AかつB)が占める割合

3. なぜ「リフト値」が重要なのか？

確信度が高いからといって、良いルールとは限りません。例えば「牛乳」が元々誰にでもよく売れる商品だった場合、「パンを買った人も当然牛乳を買う」ため、確信度が高く出ます。
「元々よく売れるから高いのか、Aを買ったから跳ね上がったのか」を見極めるのがリフト値です。

リフト値による相関の判定基準
負の相関 (避ける) < 1
独立 (関係なし) ＝ 1
正の相関 (意味あり！) ＞ 1

リフト値が「1」より大きければ大きいほど、有益な（偶然ではない）ルールと判断できます。

4. 実務での活用例（バスケット分析）

🛒 棚配置の工夫 お酒コーナーのすぐ横に、おつまみを配置してついで買いを狙う。
💻 レコメンド機能 ECサイトで「この商品を買った人は、これも買っています」と表示する。
🎟️ クーポン配布 A商品を買った顧客に、関連するB商品の割引クーポンを発行する。

試験対策まとめ

指標 何を計算している？（キーワード）
支持度 (Support) 全体に占める割合
確信度 (Confidence) Aを買った時のBの条件付き確率
リフト値 (Lift) 偶然以上か（1より大きければ有効）
