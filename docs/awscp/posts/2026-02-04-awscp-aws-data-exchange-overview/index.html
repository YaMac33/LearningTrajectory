AWS Data Exchangeの概要

#AWS #Data Exchange #データ購入 #クラウド #分析

AWS Data Exchangeは、様々なデータを簡単に購入・購読できるマーケットプレイスです。

AWS Data Exchange（データエクスチェンジ）は、一言で言うと「データの『Amazon.com（通販サイト）』」です。
世の中にある膨大なデータ（気象情報、金融データ、位置情報、医療統計など）を、「買いたい人」と「売りたい人」が安全かつスムーズにやり取りできる場所をAWSが提供しています。

1. なぜこれが必要なの？（抱えている課題）

例えば、ビールメーカーが「最新の気象データを使って、明日のビールの売上を予測したい」と考えたとします。しかし、外部データを手に入れるのは以前は非常に面倒でした。

データ調達の進化
これまでの苦労
  • ベンダーごとに個別契約 (書類地獄)
  • FTPサーバーの設定が必要
  • ハードディスクの郵送 (物理)
  • 手作業でダウンロード

分析開始まで数週間...

AWS Data Exchange
  • AWS上でポチッと購読
  • S3に自動で届く
  • 支払いはAWS利用料と合算
  • APIで自動取り込み

数分で分析開始！

2. 解決策：「クラウドでポチるだけ」

Data Exchange を使うと、データの購入から利用までの流れが劇的にシンプルになります。

データの購入フロー
🏢 データ販売者
(気象会社など)
➡
🛒 Data Exchange
カタログ検索・購読
➡
🪣 あなたのS3
自動配信・分析
  1. 見つける: AWSコンソール上の「カタログ」から、欲しいデータを検索します（無料のものから数百万円するものまであります）。
  2. 購読する: 数クリックで購読を開始します。支払いはAWSの利用料と一緒に請求されます。
  3. 使う: 購読したデータは、あなたの S3 バケットに直接配信されます。そこからすぐに Redshift や Athena で分析できます。

3. 他のサービスとの違い（ここが試験のポイント！）

「データのやり取り」に関係するサービスはいくつかありますが、対象が違います。

サービス名 役割 中身 (対象物)
AWS Data Exchange 「外部データの購入」 データセット
(気象、株価、人口統計など)
AWS Marketplace 「ソフトの購入」 AMI, SaaS, ツール
(セキュリティソフト、OSなど)
AWS Data Pipeline 「データの移動」 自社内のデータの加工・移動
Amazon AppFlow 「SaaS連携」 Salesforce等のデータをAWSへ

4. 試験対策キーワード

試験で以下の文脈が出たら、AWS Data Exchange が正解です。

  • 「サードパーティ（他社）のデータセットを簡単に見つけて購読したい」
  • 「金融、ヘルスケア、小売などの専門的なデータをAWS上で直接利用したい」
  • 「データの購読、支払い、管理を一元化したい」

まとめ

  • AWS Data Exchange: 他社が販売しているデータを、AWS上で「購入・購読」できるマーケットプレイス。
  • メリット: データの受け渡しが自動化され、S3に直接届くので、すぐに分析に回せる。
  • たとえ: 「データのApple Store / Google Playストア」。

「外からデータセットを買ってきたいな」と思ったら、このサービスの出番です！
