ビットの基本と図解：なぜ「1増えると2倍」なのか？

  • #ビット
  • #情報
  • #デジタル
  • #計算
  • #二進数

コンピュータの世界を理解する上で避けて通れないのが「ビット（bit）」です。定義としては「情報の最小単位」や「二進数の1桁（Binary Digit）」と言われますが、直感的にイメージするのは意外と難しいものです。

この記事では、計算式ではなく「箱が増えていくイメージ」を使って、ビットが増えるとなぜ情報量が爆発的に増えるのかを図解します。

ビットとは「区別できる数」のこと

ビットの本質は「0か1か」という記号そのものではなく、「いくつ区別できるか（パターンの数）」にあります。

  • 1ビット：コインの裏表（2通り）
  • 2ビット：ジャンケンの手（3通り）も余裕で表現可能（4通りまでOK）

【図解】箱が分裂して倍になる

「1ビット増える」とは、既存のすべてのパターンに対して「0の場合」と「1の場合」の2通りが追加されることを意味します。つまり、世界が2倍に広がります。

このように、ビットが1つ増えるたびに、箱（表現できるパターン）の数が倍々に増えていきます。

数式での理解：2のn乗

この増え方は「2のn乗（2ⁿ）」で表されます。

  • 1ビット：2¹ = 2通り
  • 2ビット：2² = 4通り
  • 3ビット：2³ = 8通り
  • ...
  • 8ビット：2⁸ = 256通り（これが1バイト！）

8ビット（1バイト）あれば、アルファベット（26文字）や数字、記号を含めても十分に区別できるため、文字情報の基本単位として使われています。

「分岐」としてイメージする

試験などで「nビットで表現できる数は？」と聞かれたときは、以下の「分岐ツリー」を頭に浮かべると計算しやすくなります。

Start ├─ 0 │ ├─ 00 │ │ ├─ 000 │ │ └─ 001 │ └─ 01 │ ├─ 010 │ └─ 011 └─ 1 ├─ 10 │ ├─ 100 │ └─ 101 └─ 11 ├─ 110 └─ 111

※右に行く（ビットが増える）たびに、枝が2つに分かれて倍増しています。

まとめ：試験対策ワンポイント

ビットの問題が出たら、複雑に考えずに以下のフレーズを思い出してください。

「ビットが1増えるたび、世界（表現できる数）は2倍になる」

5ビットなら？ 4ビット(16)の倍で32。
6ビットなら？ 32の倍で64。
この感覚を持っていれば、計算式を忘れても答えにたどり着けます。
