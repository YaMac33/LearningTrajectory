ITパスポートの尺度の理解

データの計算方法を理解し、正しい尺度選択が試験での得点源となる。

1行要約（超重要）

尺度とは、データをどこまで計算してよいかを決める物差し。名義 → 順序 → 間隔 → 比例の順に、扱える計算の自由度が増える。

# 尺度（Measurement Scale）とは何か

  • 定義: データをどのようなルールで数値化しているかの分類。数字そのものではなく、数字に許された意味の範囲を表す。
  • なぜ重要か: 尺度を誤ると「やってはいけない計算」を正しいと思ってしまう。「このデータ、平均していい？」「2倍って言っていい？」を見抜けるかが問われる。
  • ITパスポートの狙い: 統計の知識そのものより、計算できる／できないの判断力を見る。

# 4つの尺度（本体）

① 名義尺度（Nominal Scale）

特徴: 分類・ラベル付けのみ。数字はただの記号（意味なし）。

できること: 同じ／違うの判定

できないこと: 大小比較、足し算・平均

例: 性別（1=男性, 2=女性）、血液型、都道府県コード
※ [男性=1] と [女性=2] → 2の方が大きい？ ❌ → 平均1.5？ ❌

② 順序尺度（Ordinal Scale）

特徴: 順番・順位に意味がある。ただし「差の大きさ」は不明。

できること: 大小比較（どちらが上か）

できないこと: 差の計算、比率の計算

例: 成績（A・B・C）、満足度（高・中・低）、順位
※ A > B > C → OK、AとBの差は？ → ❌（わからない）

③ 間隔尺度（Interval Scale）

特徴: 差（間隔）に意味がある。ゼロは「基準点」であり絶対ではない。

できること: 足し算・引き算、平均

できないこと: 比率（◯倍）

例: 気温（℃）、西暦
※ 20℃ − 10℃ = 10℃ → OK、20℃は10℃の2倍 → ❌

④ 比例尺度（Ratio Scale）

特徴: 差も比も意味がある。絶対的なゼロ（＝存在しない状態）がある。

できること: 四則演算すべて

例: 売上金額、重量、長さ、年齢
※ 20歳は10歳の2倍 → OK、0円 = 何もない → OK

# 図解：扱える計算の範囲（必須暗記）

尺度 扱える計算
名義尺度 同じ／違う
順序尺度 ＋ 大小関係
間隔尺度 ＋ 差・平均
比例尺度 ＋ 比・四則演算

👉 下に行くほど「できる計算が増える」

# A vs B（試験で最重要）間隔尺度 vs 比例尺度

観点 間隔尺度 比例尺度
ゼロ 基準点（意味なし） 無（意味あり）
比 意味なし 意味あり
例 気温 年齢

20℃は10℃の2倍？ → ❌
20歳は10歳の2倍？ → ✅
👉 「ゼロに意味があるか」だけで判断できる

試験対策・覚え方

覚え方（鉄板）

名・順・間・比 （めい・じゅん・かん・ひ）

→ 下に行くほど「計算OK」

試験の引っかけポイント
  • 「平均を求めよ」→ 間隔尺度・比例尺度のみ
  • 「◯倍である」→ 比例尺度のみ
  • 数字が出てきても安心するな → 数字 ≠ 計算OK

# 理解を深めるQ&A

Q. 「順位」は差の計算ができそうなのに、なぜできない？

直感的には「1位と3位の差は2」と考えがちですが、順位は「量」ではなく「位置番号」だからです。

ケースA（接戦）: 1位(100点)、2位(99点)、3位(98点) → 順位差2、点数差2
ケースB（圧勝）: 1位(100点)、2位(60点)、3位(10点) → 順位差2、点数差90
👉 順位の差は、実際の「実力差」を保証しない。

Q. 間隔尺度の「ゼロは基準点」とは？

0℃は「熱さが何もない」わけではなく、単に氷が溶ける温度を0と決めただけです。 0円（お金がない＝存在しない）のような「絶対的な無」とは異なります。

試験用フレーズ：ゼロが動くなら、それは基準点

まとめ（超短縮）

  • 尺度 = 計算の免許証
  • ゼロに意味がある？ → 比例尺度
  • 平均できる？ → 間隔以上
  • 比を言ってる？ → 比例しか無理
