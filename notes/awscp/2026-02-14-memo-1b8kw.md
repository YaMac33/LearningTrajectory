============================================================
Lightsail / Beanstalk / Amplify の違い（図解）
============================================================

まず一言まとめ：

Lightsail  = 超シンプルなVPSセット
Beanstalk  = アプリを載せるPaaS
Amplify    = フロントエンド特化の開発基盤

------------------------------------------------------------
① Amazon Lightsail
------------------------------------------------------------
正式名称：
Amazon Lightsail

■ コンセプト
「簡単にサーバーを立てたい人向け」

■ 図

あなた
  ↓
Lightsail
  ↓
┌────────────┐
│ 仮想サーバー │
│ 固定料金     │
│ OS込み       │
└────────────┘

■ 特徴
・月額固定料金
・VPSに近い
・初心者向け
・WordPressなど簡単構築

■ 例
小規模ブログ
検証環境
個人開発

------------------------------------------------------------
② AWS Elastic Beanstalk
------------------------------------------------------------
正式名称：
AWS Elastic Beanstalk

■ コンセプト
「コードをアップすればインフラは自動構築」

■ 図

あなた（コードアップロード）
        ↓
Beanstalk
        ↓
┌────────────┐
│ EC2        │
│ ALB        │
│ AutoScaling│
│ RDS        │
└────────────┘

■ 特徴
・PaaS型
・EC2などを裏で自動構築
・スケーリング自動
・バックエンド向け

■ 例
Webアプリ
APIサーバー
業務システム

------------------------------------------------------------
③ AWS Amplify
------------------------------------------------------------
正式名称：
AWS Amplify

■ コンセプト
「フロントエンド開発を最速化」

■ 図

GitHub
   ↓
Amplify
   ↓
┌────────────┐
│ フロント配信 │
│ 認証(Cognito)│
│ API連携      │
│ CI/CD自動    │
└────────────┘

■ 特徴
・React / Vue向け
・フロント特化
・バックエンドも自動接続
・サーバーレス前提

■ 例
SPAアプリ
モバイルアプリ
スタートアップ向け

------------------------------------------------------------
全体比較図
------------------------------------------------------------

レイヤー別に見ると：

【アプリ開発支援】
        Amplify

【アプリ実行基盤（PaaS）】
        Beanstalk

【シンプル仮想サーバー】
        Lightsail


------------------------------------------------------------
難易度イメージ
------------------------------------------------------------

簡単 ←──────────────→ 柔軟

Lightsail → Amplify → Beanstalk

------------------------------------------------------------
まとめ
------------------------------------------------------------

Lightsail
 ＝ 「とりあえずサーバー」

Beanstalk
 ＝ 「コードを投げればWebアプリ完成」

Amplify
 ＝ 「フロントエンド高速開発環境」

------------------------------------------------------------
一文まとめ
------------------------------------------------------------

Lightsailは簡易サーバー、
Beanstalkはアプリ実行PaaS、
Amplifyはフロントエンド特化の開発基盤。
