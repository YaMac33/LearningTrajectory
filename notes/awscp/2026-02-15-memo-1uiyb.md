============================================================
SSH / HTTP / HTTPS / SSL の違いを図解で理解する
============================================================

まず結論：

Linux EC2へログインするのは → SSH

------------------------------------------------------------
■ ① SSH（Secure Shell）
------------------------------------------------------------

用途：
サーバーにログインしてコマンドを実行するためのプロトコル

図：

あなたのPC
     ↓（ポート22）
   SSH接続
     ↓
Linux EC2
     ↓
コマンド実行

特徴：
・暗号化されている
・鍵認証を使う（.pem）
・サーバー管理用

例：
ssh -i key.pem ec2-user@IPアドレス

------------------------------------------------------------
■ ② HTTP
------------------------------------------------------------

用途：
Webページを見るための通信

図：

ブラウザ
   ↓（ポート80）
   HTTP
   ↓
Webサーバー
   ↓
HTML返却

特徴：
・暗号化なし
・通信内容が見える
・Web通信の基本

------------------------------------------------------------
■ ③ HTTPS
------------------------------------------------------------

用途：
暗号化されたWeb通信

図：

ブラウザ
   ↓（ポート443）
   HTTPS
   ↓
Webサーバー
   ↓
暗号化通信

特徴：
・HTTP＋暗号化
・クレジットカード情報など安全
・現在のWeb標準

------------------------------------------------------------
■ ④ SSLとは？
------------------------------------------------------------

SSLは「暗号化技術」のこと。
※現在はTLSが主流（SSLの後継）

図：

HTTP
  ＋
SSL/TLS
  ↓
HTTPS

つまり：

HTTPS = HTTP + SSL/TLS

------------------------------------------------------------
■ 4つの関係をまとめると
------------------------------------------------------------

        ┌────────────┐
        │   SSL/TLS   │ ← 暗号化の仕組み
        └────────────┘
               ▲
               │
HTTP  ─────────┘ → HTTPS
（暗号化なし）        （暗号化あり）

SSHは別系統：
リモートログイン専用の暗号化プロトコル

------------------------------------------------------------
■ 用途別まとめ
------------------------------------------------------------

SSH   → サーバーにログイン
HTTP  → 暗号化なしWeb通信
HTTPS → 暗号化Web通信
SSL   → 暗号化技術そのもの

------------------------------------------------------------
■ 試験での即判断
------------------------------------------------------------

「Linux EC2へアクセス」
→ SSH

「Webページを見る」
→ HTTP / HTTPS

「暗号化通信」
→ HTTPS（中身はSSL/TLS）

------------------------------------------------------------
■ 超シンプル覚え方
------------------------------------------------------------

SSH ＝ サーバー操作用
HTTP ＝ Web閲覧
HTTPS ＝ 安全なWeb閲覧
SSL ＝ 暗号の仕組み

------------------------------------------------------------
■ 一文まとめ
------------------------------------------------------------

Linux EC2へリモートログインするにはSSHを使用し、
HTTP/HTTPSはWeb通信、SSLは暗号化技術である。
